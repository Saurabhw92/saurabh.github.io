What is the project about?

This project demonstrates the implementation of a monorepo using Nx, containing an Angular application, a React application, and a shared JavaScript library with reusable functionality. I leveraged Nx to manage the monorepo efficiently, ensuring modularity and code reusability between the applications. Additionally, I set up a GitLab CI/CD pipeline to automate the deployment of both applications to AWS S3, streamlining the build and deployment process.

Project Graph

The Nx project graph provides a visual representation of the dependencies within the monorepo. In this project, both the Angular application (angular-app) and React application (react-app) explicitly depend on the shared library (shared-lib), meaning they utilize common functions or utilities from it. Additionally, the end-to-end (E2E) test projects for each application have implicit dependenciesâ€”angular-app-e2e is implicitly dependent on angular-app, and react-app-e2e is implicitly dependent on react-app. This structure ensures modularity, code reuse, and maintainability, allowing updates in the shared library to reflect across both applications while keeping testing workflows aligned with their respective apps.

Gitlab Pipeline

The GitLab CI/CD pipeline utilizes the Nx affected commands to optimize the build and deployment process. Instead of running tasks on all projects, the pipeline intelligently detects which applications or libraries have changed and executes linting, building, and deployment only for the affected ones. This approach significantly improves efficiency by reducing unnecessary computations, leading to faster build times and streamlined deployments. By leveraging Nx's dependency graph and affected command, the pipeline ensures that only the modified parts of the monorepo are processed, making the CI/CD workflow more scalable and performant.

Technology stack

  • Monorepo Management: Nx
  • Frontend Frameworks: Angular, React
  • Shared Library: JavaScript
  • CI/CD: GitLab CI/CD
  • Cloud Storage & Deployment: AWS S3
